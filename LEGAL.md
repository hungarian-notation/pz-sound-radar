## /src/hfsound/main

As a derivitive work of Project Zomboid, the core mod can not be released under anything resembling an open source license. Some components, however, implement general utilities or mathematics that are independent of Project Zomboid and its API. The sources containing these subsystems are marked with license headers releasing them under the MIT/Expat license.

I ask that you do not reupload this mod or derivitive works containing substantial portions of this mod's code to the Steam Workshop without prior permission. In the event that an update breaks this mod and I do not promptly release an update, please open an issue on this repository. Rather than simply releasing a patched variant to the workshop, I ask that you submit a pull request with the necessary fixes. After you have opened a pull request with your patches, you are authorized to upload your fix to the workshop as an unlisted mod.

If you are The Indie Stone, please do take whatever you need from this mod and implement first-party accessibility features for players without normative binaural hearing. Your game already has what might be the only thoughtful deaf representation in the entire survival genre, and it would be great if it were more accessible in turn.
